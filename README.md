## Paper code

Code for all papers is kept within the namespace for that paper as `+Author_Year`

Data to produce all figures are kept in relevant OSF project as `Author Year` at this address: https://osf.io/user/pkh5w
Data on OSF was packaged using the code namespace `+Author_Year.package`

Code to reproduce paper figures are in scripts `+Author_Year.figures.generate_figures`
Note that data paths will need to be changed to point to locally downloaded data
This code may also have external dependencies, which are listed in the comments at the top of the figure script
